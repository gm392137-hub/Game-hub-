# 🎮 Game Hub

एक सुंदर Game Store ऐप जो Amazon की तरह काम करता है। यह ऐप Flutter से बनाया गया है और iOS व Android दोनों पर काम करता है।

## ✨ Features

✅ **Home Screen** - सभी games की list
✅ **Search** - किसी भी game को खोजें
✅ **Categories** - विभिन्न categories में games देखें
✅ **Game Details** - प्रत्येक game की detailed information
✅ **Rating & Reviews** - Games की rating देखें
✅ **Download/Install** - Games को install करें
✅ **Beautiful UI** - Modern और responsive design

## 📦 Installation

### Requirements
- Flutter SDK (3.0 या नया)
- Dart SDK
- Android Studio या Xcode

### Steps

1. **Clone करें:**
```bash
git clone https://github.com/gm392137-hub/Game-hub-.git
cd Game-hub-
```

2. **Dependencies install करें:**
```bash
flutter pub get
```

3. **Run करें:**
```bash
flutter run
```

## 🏗️ Project Structure

```
lib/
```
